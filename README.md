# Smart Tip Calculator

A modern, real-time **Tip Calculator Web Application** built using **HTML, CSS, and JavaScript**.  
This project demonstrates **clean UI design**, **live calculations**, and **robust input handling**, making it suitable for academic projects and frontend portfolios.

---

## Features

- Real-time tip calculation (no submit button)
- Tip percentage slider (0% – 100%)
- Bill splitting for multiple people
- Smart rounding options (Exact / ₹1 / ₹5)
- Modern Glassmorphism UI
- Fully responsive design
- Instant UI updates using JavaScript events
- Accurate and safe arithmetic logic

---

## Technologies Used

- **HTML5** – Semantic structure
- **CSS3** – Flexbox, gradients, glassmorphism
- **JavaScript (ES6)** – DOM manipulation, event handling
